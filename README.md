## Design Philosophy

- **Auto-triggering**: Each skill has clear USE when / DO / NOT for / Triggers metadata
- **Composable**: Skills can be chained (e.g., brainstorming → writing-plans → tdd → review)
- **Lightweight**: Pure markdown skills, no runtime dependencies
- **Proven**: Adapted from top open-source projects (Superpowers, mattpocock/skills, NVIDIA SkillSpector, Microsoft SkillOpt)
