# Systematic Debugging

## 4-Phase Root Cause Analysis

1. Reproduce - confirm the issue
2. Isolate - narrow down scope
3. Locate - find root cause
4. Fix + Verify - fix and confirm

## Defense

After fixing, add guardrails at entry points to prevent similar issues.
