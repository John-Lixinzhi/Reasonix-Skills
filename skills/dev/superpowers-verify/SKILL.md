# 完成前验证

**在声称任何任务"完成"或"已修复"之前，必须运行验证命令并确认输出。**

## 验证清单

### 代码变更后
- [ ] 构建/编译是否通过？
- [ ] 所有已有测试是否通过？
- [ ] 新增测试是否通过？
- [ ] 是否有 lint/类型错误？

### Bug 修复后
- [ ] 修复前的复现步骤是否不再触发？
- [ ] 边界条件是否也处理了？
- [ ] 相关功能是否有回归？

### 重构后
- [ ] 行为是否与重构前一致？
- [ ] 测试是否全部通过？
- [ ] 是否有死代码遗留？

## 规则
- 不要凭"我觉得应该可以"就标记完成
- 把验证命令的输出贴出来作为证据
- 如果验证失败，先修复再声称完成
