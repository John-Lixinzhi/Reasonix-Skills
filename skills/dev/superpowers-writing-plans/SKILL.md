# Writing Implementation Plans

Break approved designs into small, executable tasks.

## Each Task Should

- Take 2-5 minutes
- Specify exact files
- Include verification steps
- Have clear completion criteria

## Order

- Data models first, then UI
- Dependencies before dependents
