# 会话交接 (Handoff)

来源：mattpocock/skills (140k⭐)

把当前对话压缩为交接文档，让另一个AI/会话无缝继续工作。

## 交接文档包含
- 当前状态：做了什么，做到哪了
- 决策记录：为什么这么做
- 待办事项：接下来要做什么
- 已知问题：当前遇到的坑
- 文件变更：改了哪些文件
- 领域术语：项目专用词汇表

## 使用场景
- 从一个AI切换到另一个AI（如Claude Code → Codex）
- 长任务需要多个会话完成
- 团队成员接力工作
