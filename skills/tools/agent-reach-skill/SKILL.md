# Agent-Reach — 全网数据采集

来源：Panniantong/Agent-Reach (37.6k⭐)

给你的 AI Agent 一键装上互联网采集能力。

## 支持平台
- **零配置即用**：网页/Jina Reader、YouTube字幕/搜索、RSS订阅、GitHub仓库、V2EX、B站
- **配置后解锁**：Twitter(搜索+时间线)、小红书(搜索+阅读)、Reddit、LinkedIn、雪球、小宇宙播客
