# PPT 数据可视化

来源：PPT日记视频教程技能库

## 1. 创意数据条（进度条）
- 形状拼接法：灰色底条+彩色进度条叠加
- 动画进度条：擦除动画→自左侧
- 同一系列颜色保持一致

## 2. 环形图（甜甜圈图）
- 默认图表法：插入饼图→环形图→调整内径
- 形状法：圆环剪除+弧形扇区
- 扇区不超过6个，最大扇区从12点方向开始

## 3. 柱状图美化
- 删除多余元素（网格线、图例、坐标轴）
- 柱子圆角+统一主题色+渐变填充
- 最大值用强调色突出
- 添加数据标签和趋势线

## 4. 信息图示（SmartArt）美化
- 选择合适的图示类型（流程/循环/层次/矩阵）
- 更改颜色+样式
- 转换为形状后完全自由编辑
- 每个项目前加对应图标
