# PPT 动态图文排版

来源：PPT日记视频教程技能库

## 1. 平行四边形动态排版
- 图片铺满背景，插入平行四边形（倾斜15-30度）
- 渐变填充（白色透明度20%，角度20度）
- 复制2-3份形成层次感
- 配合平滑切换实现形状滑动入场

## 2. 菱形创意排版
- 按住Shift绘制正菱形
- 图片/纯色/渐变填充，细边框或无边框
- 四象限/环形布局排列
- 配合缩放旋转动画动态展示

## 3. 渐变文字效果
- 方法一：PPT内置渐变填充（保留可编辑性）
- 方法二：文字转形状后渐变（精度更高）
- 渐变颜色控制在2-3种，仅用于标题/数据

## 4. 文字倒影效果
- 方法一：内置倒影效果（快速）
- 方法二：复制→垂直翻转→透明度20-30%→柔化边缘2-4磅
- 适合封面、产品展示、科技感PPT
