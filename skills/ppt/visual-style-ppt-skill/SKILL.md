# 视觉风格 PPT (visual-style-ppt-skill)

从**视觉风格驱动**的 PPT 生成工作流。适合需要统一品牌视觉、从参考图提取设计DNA的场景。

## 核心理念
- 设计优先：先定视觉风格，再填充内容
- 品牌一致：从参考图提取配色/字体/布局规则，保存为 Design DNA
- 多种输出：支持 HTML + PPTX 双格式

## 8 种视觉风格
1. 极简商务 — 干净、专业、留白多
2. 科技感 — 深色背景、霓虹色、未来感
3. 杂志风 — 大字标题、图片优先、版面感
4. 数据驱动 — 图表突出、KPI大字报、信息图
5. 品牌统一 — 企业VI色、固定模板、规范严格
6. 手绘/涂鸦风 — 非正式、创意、有机形状
7. 日式/极简 — 和风、侘寂、低饱和度
8. 复古/怀旧 — 暖色调、肌理感、老式排版
