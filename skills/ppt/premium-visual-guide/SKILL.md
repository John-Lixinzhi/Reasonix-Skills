# 高级视觉PPT设计指南

不是新的工具，而是**告诉你怎么做才能高级**。结合 guizang-ppt-skill + ppt-master 使用。

## 一、高级感的底层逻辑

高级 = 克制 + 一致 + 质感

| 低级感 | 高级感 |
|--------|--------|
| 多用颜色 | 主色 ≤ 2种 + 辅色 ≤ 2种 |
| 多种字体混搭 | 最多 2 种字体家族 |
| 花哨渐变/阴影 | 纯色/微渐变/无阴影 |
| 密集信息 | 大量留白 |
| 装饰性元素堆砌 | 每个元素有功能目的 |

## 二、配色系统（不能随便选）

### 高级配色公式
- **主色** (60%)：中性色（黑/白/灰/米色/深蓝）
- **辅色** (30%)：同一色系的深浅变化
- **强调色** (10%)：一个高饱和色做视觉锚点

### 永远不会错的高级配色
- 黑 + 白 + 金（奢华）
- 深蓝 + 白 + 银（科技）
- 米色 + 深棕 + 墨绿（自然/人文）
- 炭灰 + 白 + 克莱因蓝（现代）
- 深紫 + 金 + 白（创意/高端）

### 禁忌
- 荧光色做背景
- 超过 4 种独立色
- 纯黑背景纯白文字（眼睛累）
- 高饱和色做背景

## 三、背景设计（Gemini的短板）

### 高级背景类型
1. **微纹理** — subtle-noise / paper-texture / linen（比纯色有质感10倍）
2. **深色微渐变** — #0a0a0a → #1a1a2e（比纯黑有层次）
3. **几何装饰** — 极细线条/点阵/网格（比大面积渐变高级）
4. **大图+暗罩** — 高清图片 + 60-70%黑色叠加层
5. **纯色+底部光晕** — 纯色背景+底部径向渐变

### 背景禁忌
- ❌ 默认白背景 + 默认黑字（像Word文档）
- ❌ 全页渐变（像2000年代PPT）
- ❌ 带水印/低清图片
- ❌ 色彩过于鲜艳的图片做背景
- ❌ 纹理过于明显的材质（像墙纸）

## 四、字体搭配

### 高级字体组合
| 场景 | 标题字体 | 正文字体 |
|------|---------|---------|
| 科技/现代 | Inter / Helvetica Neue | Inter / Noto Sans SC |
| 人文/杂志 | Playfair Display / Noto Serif SC | Source Sans / Noto Sans SC |
| 创意/设计 | GT Walsheim / system-ui | 同标题或 Inter |
| 正式/商务 | IBM Plex Sans | IBM Plex Sans |

### 字号层级
- 封面主标题：5-7vw
- 章节标题：2.5-4vw
- 正文：1-1.5vw（最小 16px）
- 注释/来源：0.7-0.9vw（最小 12px）

## 五、留白法则
- 页面边缘留白 ≥ 5%
- 元素间距 ≥ 元素本身大小的 50%
- 文字行高 1.5-1.8 倍
- 段落间距 ≥ 行高 2 倍
- **越高级的东西留白越多**

## 六、图片调性
- 统一风格：全部纪实摄影 / 全部插画 / 全部3D渲染
- 不要混搭：不要把手机截图和单反照片放一起
- 图片必须高分辨率（≥ 1600px宽）
- 图片色调整体统一（暖调/冷调/低饱和）
- 剪裁统一比例（全部 16:9 或全部 4:3）

## 七、单页布局检查清单
- [ ] 视觉重心只有一个
- [ ] 文字够大（投影用最小 16px）
- [ ] 对比度够（文字和背景）
- [ ] 没有多余装饰元素
- [ ] 留白充分
- [ ] 配色一致
- [ ] 字体不超过 2 种
- [ ] 图片清晰且风格统一
