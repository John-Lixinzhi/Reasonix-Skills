# 抖音知识号风格PPT指南

来源：高级视觉动效制作指南

## 核心特征
- 深色背景 + 亮色点缀（高对比度、高饱和度）
- 粗体无衬线（思源黑体 Heavy / Montserrat Bold）
- 大字报式排版，大量留白
- 全屏高清大图，电影级质感
- 动画快速有力有节奏

## 页面模板

### 封面页
- 全屏高清图（暗化70%）+ 居中72pt+粗体标题
- 动画：背景淡入(0.5s)→标题飞入(0.8s)→副标题淡入(0.5s)

### 金句页
- 深色纯色背景 + 大号引号装饰（透明度20%）
- 48pt+粗体金句，16pt来源
- 动画：引号淡入→金句逐字飞入→来源淡入

### 数据展示页
- 60%图片 + 40%数据卡片（圆角+主题色边框）
- 数字48pt粗体，动画：图片滑入→卡片依次飞入→数字计数增长

### 过渡页
- 动态渐变/视频背景 + 120pt超大章节号
- 标题从下方飞入 + 进度线擦除

## 动效QA清单

- [ ] 平滑结束0.3-0.5秒？

- [ ] 字体标题≥48pt，正文≥24pt？

- [ ] 是否有足够留白？
- [ ] 整体风格统一？
