# 怀疑性认知六原则

来源：《真相：信息超载时代如何知道该相信什么》

## 六条原则

### 1. 我碰到的是什么内容？
- 这是新闻、观点、分析、广告、还是娱乐？
- 内容的类型决定了你的阅读方式
- 不同类型有不同的事实标准

### 2. 信息完整吗？假如不完整，缺少了什么？
- 有没有关键信息被省略？
- 上下文是否被截断或曲解？
- 谁受益于这种"不完整"？

### 3. 信源是谁/什么？我为什么要相信他们？
- 信源的资质、立场、利益关系是什么？
- 是一手信源还是二手转述？
- 信源过去的历史记录如何？

### 4. 提供了什么证据？是怎样检验或核实的？
- 是事实数据、引用来源、还是个人观点？
- 证据可以独立验证吗？
- 有没有提供验证方法？

### 5. 其他可能性解释或理解是什么？
- 还有没有其他合理的解释？
- 有没有相反的证据被忽略？
- 如果用不同立场看，结论会变吗？

### 6. 我有必要知道这些信息吗？
- 这条信息对我有什么用？
- 不知道会有什么后果？
- 值得花时间深入吗？

## 适用场景
- 阅读公众号/新闻时快速判断可信度
- 查文献时评估论文质量
- 收到AI输出结果时做事实核查
- 做研究时筛选信息来源
