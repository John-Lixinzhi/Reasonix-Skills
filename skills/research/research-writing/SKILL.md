# 科研文献综述与写作

基于 Scientific Agent Skills 核心工作流提炼，不依赖外部包，纯知识驱动。

## 文献检索
- 多数据库并行检索思路：PubMed/arXiv/bioRxiv/Semantic Scholar
- 自动去重、带DOI和摘要返回
- 检索结果按主题分组，提炼核心发现
- 标记不同研究之间的结论差异
- 输出：有组织的阅读路线图

## 学术写作（IMRAD结构）
- **Introduction**：研究背景、问题陈述、假设
- **Methods**：方法描述、实验设计
- **Results**：结果呈现、数据分析
- **Discussion**：结果解读、局限性、未来工作
- 辅助检查措辞、结构、逻辑连贯性

## 引用管理
- BibTeX 格式自动生成
- 兼容 Zotero
- 支持常见引用格式（APA/MLA/Chicago等）

## 数据库查询
- 提供 78 个公开科学数据库的统一查询思路
- 根据研究领域选择对应数据库
- 结构化返回查询结果

## 工作流
1. 确定研究问题 → 拆解关键词
2. 多数据库并行检索 → 去重整理
3. 文献分组 → 提取核心发现
4. 写作架构 → IMRAD填充
5. 引用管理 → 生成BibTeX
