# Frontend Design — 前端UI/UX设计

从设计系统出发，生成风格一致的前端 UI。

## 核心流程
1. **理解需求** — 页面类型/目标用户/品牌调性/功能清单
2. **设计系统匹配** — 配色/字体/间距/圆角/阴影规范
3. **布局规划** — 信息层级/栅格系统/响应式断点
4. **组件设计** — 按钮/表单/卡片/导航/弹窗等
5. **页面组装** — 将组件组合成完整页面
6. **动效设计** — 过渡/悬停/加载状态

## 设计原则
- 一致性优于创意
- 内容优先，装饰克制
- 移动优先，渐进增强
- 无障碍设计（a11y）内置
